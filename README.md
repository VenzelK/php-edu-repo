# PHP Base

## Setup

### For enable env select .env.* for needed version:

```bash example
cp .env.prod .env
```

```bash example
cp .env.dev .env
```

#### For start docker-compose containers:

```bash
docker-compose up -d
```

## Usage


Goto http://127.0.0.1:3000 or http://127.0.0.1:443 for prod
